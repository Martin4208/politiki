# POLITIKI

あなたの政治的価値観を診断し、日本の主要政党との一致度を可視化するWebアプリです。

## 概要

いくつかの質問に答えるだけで、自分の政治プロフィールを確認できます。各政党の政策スタンスとの比較や、政策ごとの各党の立場を学ぶことができます。

## 主な機能

- **政治診断** — 質問に答えて自分の政治的傾向を把握
- **政党比較** — 回答結果をもとに各政党との一致度をスコア表示
- **政策スタンス** — 各政党が各政策に対してどのような立場を取っているか確認
- **政党詳細ページ** — 政党ごとの理念・政策スタンスを詳細表示
- **ゲストモード** — アカウント不要で診断可能（結果はブラウザ内のみ保存）
- **アカウント機能** — 登録・ログイン・パスワードリセット

## セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/ユーザー名/リポジトリ名.git
cd リポジトリ名
```

### 2. 依存関係をインストール

```bash
npm install
```

### 3. 環境変数を設定

`.env` ファイルをプロジェクトルートに作成し、以下を設定:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
```

Supabaseのプロジェクト設定 → API から取得できます。

### 4. 開発サーバーを起動

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) で確認できます。
